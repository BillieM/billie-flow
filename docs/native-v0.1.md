# Billie Flow Native v0.1 (historical)

Native v0.1 was the private implementation and automated-acceptance milestone
that proved the Swift app, persistent local worker, microphone path, clipboard
policy, and production model latency. Its release build required the worker to
be installed separately from a repository checkout.

The public proof-of-concept release supersedes that setup with an Apple
Silicon-only app, explicit in-app download consent, and a bundled bootstrap
payload. See `docs/native-v0.2.md` for current installation, model attribution,
privacy, storage, Gatekeeper, and uninstall documentation. The original v0.1
verification evidence remains in `docs/native-v0.1-qa.md`.
