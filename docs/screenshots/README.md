# Billie Flow screenshots

These were captured from the installed v0.2.1 release candidate on macOS 26.
Each PNG contains only a Billie Flow window or HUD: the cursor, desktop, other
apps, notifications, and machine-identifying chrome are excluded.

Recommended blog sequence:

1. `billie-flow-first-run-settings.png` — the narrow settings surface, fixed
   cleanup style, temporary-audio promise, and local-worker section.
2. `billie-flow-install-consent.png` — the explicit 3.5 GB Hugging Face model
   download, English-only, Apple Silicon, and macOS 26 disclosure.
3. `billie-flow-downloading-speech-model.png` — setup downloading the fixed
   local speech model without a separate Terminal command.
4. `billie-flow-install-complete.png` — the local speech and cleanup models are
   installed and ready to load.
5. `billie-flow-recording.png` — the non-activating hold-to-record HUD.
6. `billie-flow-copied.png` — a real local ASR and light-cleanup run completed
   successfully.

`billie-flow-installed-local-worker.png` is an alternate, tighter crop of the
completed setup state.
